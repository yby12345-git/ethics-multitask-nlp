# Ethics Multi-task NLP

## Overview
This repository contains the code for a multi-task BERT framework for ethical text classification and interpretability analysis.

## Environment
- Python 3.10
- PyTorch
- Transformers

## Project Structure
- `scripts/`: training and evaluation scripts
- `plot_figure5_clean.py`: script for generating Figure 5
- `Figure5_clean.png` / `Figure5_clean.pdf`: generated figure files
